如何使用hugo搭建一个个人博客？

17 May, 2020 · Read in about 3 min · (485 Words)
hugo

1.下载二进制安装包文件

Hugo二进制下载地址：https://github.com/spf13/hugo/releases

2.解压安装包

cd /usr/local
mkdir hugo
cd hugo
tar -xf hugo_0.70.0_Linux-64bit.tar.gz

3.创建网站

cd /usr/local/hugo
./hugo new site

然后hugo会自动生成这样一个目录结构：

  ▸ archetypes/
  ▸ content/
  ▸ layouts/
  ▸ static/
    config.toml

4.下载主题

$ cd /usr/local/hugo/site
$ cd themes
$ git clone https://github.com/enten/hyde-y

5.修改配置文件

1)切换目录

cd /usr/local/hugo/site

2)修改配置文件config.toml

[root@gavin site]# more config.toml 
# hostname (and path) to the root eg. http://spf13.com/
baseurl = "https://troub1emaker0911.github.io"

# Site title
title = "Gavin Stone's Blog"

# Copyright
copyright = "(c) 2020 Gavin Stone."

# Language
#languageCode = "en-EN"
languageCode = "zh-CN"

# Metadata format
# "yaml", "toml", "json"
metaDataFormat = "yaml"

# Theme to use (located in /themes/THEMENAME/)
theme = "hyde-y"

# Pagination
paginate = 10
paginatePath = "page"

# Enable Disqus integration
disqusShortname = "Gavin"

[permalinks]
    post = "/:year/:month/:day/:slug/"
    code = "/:slug/"

[taxonomies]
    tag = "tags"
    topic = "topics"

[author]
    name = "Gavin Stone"
    email = "xialiaoliao0911@gmail.com"

#
# All parameters below here are optional and can be mixed and matched.
#
[params]
    # You can use markdown here.
    brand = "Gavin Stone"
    topline = "welcome to Gavin Stone's Blog"
    footline = "code with <i class='fa fa-heart'></i>"

    # Sidebar position
    # false, true, "left", "right"
    sidebar = "left"

    # Text for the top menu link, which goes the root URL for the site.
    # Default (if omitted) is "Home".
    home = "首页"

    # Select a syntax highight.
    # Check the static/css/highlight directory for options.
    highlight = "default"

    # Google Analytics.
    googleAnalytics = "Your Google Analytics tracking code"

    # Sidebar social links.
    github = "enten/hugo-boilerplate" # Your Github profile ID
    bitbucket = "" # Your Bitbucket profile ID
    linkedin = "" # Your LinkedIn profile ID (from public URL)
    googleplus = "" # Your Google+ profile ID
    facebook = "" # Your Facebook profile ID
    twitter = "" # Your Twitter profile ID
    youtube = ""  # Your Youtube channel ID
    flattr = ""  # populate with your flattr uid

[blackfriday]
    angledQuotes = true
    fractions = false
    hrefTargetBlank = false
    latexDashes = true
    plainIdAnchors = true
    extensions = []
    extensionmask = []

3)新建配置文件FootMenu.toml和Menu.toml

cd /usr/local/hugo/site/data

[root@gavin data]# pwd
/usr/local/hugo/site/data
[root@gavin data]# more FootMenu.toml 
[license]
    Name = "授权"
    URL = "/license"

[root@gavin data]# pwd
/usr/local/hugo/site/data
[root@gavin data]# more FootMenu.toml 
[license]
    Name = "授权"
    URL = "/license"
[root@gavin data]# more Menu.toml 
[about]
    Name = "关于我"
    URL = "/about"

[posts]
    Name = "文章列表"
    Title = "Show list of posts"
    URL = "/post"

[tags]
    Name = "标签"
    Title = "Show list of tags"
    URL = "/tags"

4)新建一篇文章,一般文档都存储在/usr/local/hugo/content/post目录下。

cd /usr/local/hugo/site
./hugo new post/first.md

这样就在/usr/local/hugo/site/content/post目录下新建了一篇文章first.md。但要使你的文章可见，需要打开文件first.md，将draft: true 修改为draft: false. 具体如下：

[root@gavin post]# more first.md 
---
title: "First"
date: 2020-05-12T23:51:02+08:00
draft: false
---


### Hello Hugo

 1. aaa
 1. bbb
 1. ccc

6.生成静态页面

cd /usr/local/hugo/site
./hugo --theme=hyde-y   --baseUrl="http://troub1emaker0911.github.io/"

7.托管网站到Github

1)在Github上新建一个仓库，名称为：troub1emaker0911.github.io 这里的troub1emaker0911为你的Github用户名。

2)上传代码到你的Github仓库

cd /usr/local/hugo/site/public
echo "# troub1emaker0911.github.io" >> README.md
git init
git add README.md
git add -A
git commit -m "first commit"
git remote add origin https://github.com/troub1emaker0911/troub1emaker0911.github.io.git
git push -u origin master
